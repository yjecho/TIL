## Terminal commands

* date
* . *vs* ..
* Listing Directory
  * ls
  * ls -a
* Change Directory
  * cd
* Clear the Terminal screen of all previous commands
  * clear
* Create a text file
  * touch a.txt
* Create a folder
  * mkdir test 
* Move a file
  * mv a.txt b.txt
  * mv b.txt test2
* Remove files or directories
  * rm a.txt
  * rm -r test
  * rm *.txt
  * rm -rf
  * rm -rf*
  * rm -r .git
